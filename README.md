# VKMessageDumper

VKMessageDumper — консольная утилита на Python для выгрузки фотографий из диалогов и бесед ВКонтакте.

Поддерживает:
- личные диалоги
- групповые беседы  
Не поддерживает:
- диалоги с ботами и сообществами

Проект рассчитан на обычных пользователей: достаточно вставить запрашиваемые ссылки, всё остальное делается автоматически.

---

## Возможности

- Загрузка всех фотографий из диалога
- Сохранение в максимальном качестве
- Многопоточная загрузка
- Шкала прогресса
- Продолжение — можно прервать и продолжить
- OAuth-токен сохраняется локально
- Автоматическая проверка валидности токена
- Принимает ссылки, без ручного ввода ID

---

## Требования

- Python **3.9+**
- ОС: Windows / Linux / macOS

## Установка
Скачать архив из релиза или клонировать репозиторий:

```bash
git clone https://github.com/DeezLikerStudio/VKMessageDumper.git
cd VKDumper
python VKDumper.py
```

---

## Как пользоваться

1) Перейти по ссылке https://oauth.vk.com/authorize?client_id=6287487&display=page&redirect_uri=https://oauth.vk.com/blank.html&scope=messages,photos&response_type=token&v=5.131
2) Авторизоваться, откроется пустое окно. Скопировать всю ссылку из адрессной строки в программу (OAuth URL). P.S. Токен живёт сутки, сохраняется локально. При невалидности программа запросит новый токен;
3) Вставить ссылку на беседу/переписку;
4) Назвать папку для сохранения фотографий. Папка создастся в папке со скриптом;
5) Ждать.
